# agent-maaz

> Free open-source AI agent. $0 recurring cost, multi-layer LLM fallback, web + terminal UI.

## Vision

Free for the people who can't afford subscriptions. Independent — no corporate control. Best-of-breed features from the strongest open-source agents, with free-tier models across multiple providers.

## Status

🚧 **v0.1.0 — early development.** Personal testing phase. Will be open-sourced publicly when stable.

## Requirements

- **Python 3.9+** (tested on 3.9.6)

## Stack

- **Python 3.9+** backend
- **OpenRouter** (free tier, DeepSeek V3.1 as primary)
- **LiteLLM** (local, multi-provider fallback)
- **Ollama** (local safety net)
- **Open WebUI** (web UI)
- **textual** (terminal UI)
- **FastAPI** (backend server)
- **Chroma** (local memory)
